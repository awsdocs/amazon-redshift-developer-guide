# PG\_DATABASE\_INFO<a name="r_PG_DATABASE_INFO"></a>

PG\_DATABASE\_INFO is an Amazon Redshift system view that extends the PostgreSQL catalog table PG\_DATABASE\. 

PG\_DATABASE\_INFO is visible to all users\.
